# 🔮 MZ 스타일 관상 분석기

## 🌟 프로젝트 개요

이 프로젝트는 **OpenCV**를 사용하여 웹캠 비디오 스트림을 실시간으로 표시하고, **Tkinter** 기반의 GUI를 통해 사용자에게 MZ 세대가 흥미를 느낄 만한 재미있는 관상 분석 결과를 시뮬레이션하여 제공합니다. 웹캠 연결, 실시간 피드 출력, 그리고 다중 스레딩을 활용한 분석 시뮬레이션 기능을 구현했습니다.

### 주요 기능

* **웹캠 스트리밍:** OpenCV를 이용한 실시간 비디오 피드 표시 (거울 모드).
* **다중 스레딩:** 비디오 루프와 GUI 이벤트를 분리하여 프로그램 멈춤(Freezing) 현상을 방지.
* **분석 시뮬레이션:** "관상 분석 시작" 버튼 클릭 시 3초간의 분석 시뮬레이션 후 랜덤 결과 표시.
* **Tkinter GUI:** 사용자가 쉽게 조작할 수 있는 직관적이고 시각적인 인터페이스 제공.
